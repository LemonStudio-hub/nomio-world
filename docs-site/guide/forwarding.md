# 邮件转发

本章介绍邮件转发的配置和使用。

## 什么是邮件转发？

邮件转发功能可以将收到的邮件自动转发到你指定的外部邮箱。

## 配置转发

### 1. 进入邮箱设置

登录后，进入邮箱页面，点击"设置"标签。

### 2. 填写转发邮箱

在"转发邮箱"字段填写你的外部邮箱地址。

### 3. 保存设置

点击"保存设置"按钮。

## 转发邮箱

### 支持的邮箱

- Gmail
- Outlook
- Yahoo
- QQ 邮箱
- 163 邮箱
- 其他任何邮箱

### 示例

| 转发邮箱 | 说明 |
|----------|------|
| your@gmail.com | Gmail |
| your@outlook.com | Outlook |
| your@qq.com | QQ 邮箱 |

## 启用/禁用转发

### 启用转发

1. 进入邮箱设置
2. 勾选"启用邮箱功能"
3. 填写转发邮箱
4. 保存设置

### 禁用转发

1. 进入邮箱设置
2. 取消勾选"启用邮箱功能"
3. 保存设置

## 转发说明

### 转发时机

- 收到邮件后立即转发
- 转发与接收同步进行

### 转发内容

- 转发完整的邮件内容
- 包括附件
- 保留原始发件人信息

### 转发限制

- 每封邮件只转发一次
- 转发失败不重试
- 转发不影响原始邮件存储

## 转发状态

### 查看转发状态

在邮箱设置页面可以查看：
- 是否启用转发
- 转发邮箱地址
- 存储用量

### 转发失败

转发失败可能的原因：
- 转发邮箱地址错误
- 转发邮箱拒收
- 网络问题

## 常见问题

### 转发有延迟吗？

通常没有延迟，收到邮件后立即转发。

### 转发会丢失邮件吗？

不会。转发是在接收邮件的基础上进行的，原始邮件仍然保存在 Namio.World。

### 可以转发到多个邮箱吗？

当前只支持转发到一个邮箱。如需转发到多个邮箱，可以使用外部邮箱的转发功能。

### 转发邮箱可以修改吗？

可以随时修改转发邮箱地址。

### 停止转发后，之前的邮件还在吗？

是的，停止转发只影响新收到的邮件，之前收到的邮件仍然保存在 Namio.World。
