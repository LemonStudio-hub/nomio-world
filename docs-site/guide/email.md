# 使用邮箱

本章介绍如何注册和使用 Nomio.World 邮箱。

## 注册邮箱

### 1. 进入邮箱页面

登录后，点击左侧导航栏的"邮箱"。

### 2. 注册邮箱

1. 点击"注册邮箱"
2. （可选）填写转发邮箱
3. 点击"注册邮箱"按钮

### 3. 注册成功

注册成功后，你将获得 `your-name@nomio.world` 邮箱地址。

## 邮箱功能

### 接收邮件

任何发送到 `your-name@nomio.world` 的邮件都会被接收并存储。

### 邮件转发

可以配置转发邮箱，收到的邮件会自动转发到指定邮箱。

### 存储空间

每个用户有 100MB 的邮件存储配额。

## 接收邮件

### 邮箱地址

你的邮箱地址是：`your-name@nomio.world`

### 接收限制

- 单封邮件最大 5MB
- 同一发件人 5 分钟内最多 3 封
- 连续 90 天未登录将冻结邮箱

### 查看邮件

1. 进入邮箱页面
2. 点击"收件箱"标签
3. 点击邮件查看详情

## 邮件客户端

Nomio.World 提供了专业的邮件客户端功能：

### 邮件列表

- 分页显示
- 搜索功能（搜索发件人、主题、正文）
- 筛选功能（全部、未读、已读、星标）
- 排序功能（按时间、大小、发件人、主题）

### 邮件详情

- 纯文本和 HTML 显示
- HTML 邮件净化（防止 XSS）
- 星标标记
- 已读/未读状态

### 批量操作

- 批量删除
- 批量标记已读
- 批量标记未读

## 邮件转发

### 配置转发

1. 进入邮箱页面
2. 点击"设置"标签
3. 填写转发邮箱
4. 点击"保存设置"

### 转发说明

- 开启后，收到的邮件会同时转发到指定邮箱
- 转发邮箱可以是任何邮箱（如 Gmail、Outlook）
- 留空则不转发

### 启用/禁用

可以随时启用或禁用邮箱功能：
- 启用：正常接收邮件
- 禁用：停止接收邮件

## 邮箱设置

### 设置选项

| 设置 | 说明 |
|------|------|
| 启用邮箱 | 开启/关闭邮箱功能 |
| 转发邮箱 | 邮件转发地址 |

### 存储用量

在设置页面可以查看存储用量：
- 已用空间
- 总配额（100MB）
- 使用百分比

## 常见问题

### 邮箱可以发送邮件吗？

当前版本仅支持接收邮件，不支持发送。你可以在 Gmail 等客户端中设置"以该地址代发"。

### 邮件存储有容量限制吗？

每个用户有 100MB 的邮件存储配额。超出后自动删除最早的邮件。

### 如何转发邮件到其他邮箱？

在邮箱设置中填写转发邮箱地址，收到的邮件将自动转发到该邮箱。

### 域名可以自定义吗？

当前仅支持 `@nomio.world` 邮箱。自定义域名邮箱支持计划在未来版本中提供。

### 邮箱多久生效？

注册后邮箱立即生效。可以立即接收邮件。
