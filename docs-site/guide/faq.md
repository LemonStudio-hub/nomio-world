# 常见问题

## 域名相关

### 域名多久生效？

注册成功后域名立即生效。Cloudflare DNS 全球传播通常在几分钟内完成。

### 支持哪些源站？

支持任何支持 HTTPS 的源站，包括 Vercel、Netlify、GitHub Pages、自有服务器等。不支持 IP 地址。

### 域名可以自定义吗？

当前仅支持 `*.nomio.world` 子域名。自定义域名支持计划在未来版本中提供。

### 如何查看域名状态？

在域名管理页面可以查看域名的验证状态。

### 域名被封禁怎么办？

如果域名被封禁，请联系管理员了解原因并申请解封。

## 邮箱相关

### 邮箱可以发送邮件吗？

当前版本仅支持接收邮件，不支持发送。你可以在 Gmail 等客户端中设置"以该地址代发"。

### 邮件存储有容量限制吗？

每个用户有 100MB 的邮件存储配额。超出后自动删除最早的邮件。

### 如何转发邮件到其他邮箱？

在邮箱设置中填写转发邮箱地址，收到的邮件将自动转发到该邮箱。

### 域名可以自定义吗？

当前仅支持 `@nomio.world` 邮箱。自定义域名邮箱支持计划在未来版本中提供。

### 邮箱多久生效？

注册后邮箱立即生效。可以立即接收邮件。

### 如何查看邮件大小？

在邮件列表页面可以查看每封邮件的大小。

## 账号相关

### 忘记密码怎么办？

目前暂不支持密码重置功能。如果忘记密码，请联系管理员。

### 如何修改用户名？

用户名注册后不可修改。如需更换用户名，请注册新账号。

### 如何删除账号？

在设置页面可以删除账号。删除后所有数据将永久删除，无法恢复。

### 账号被冻结怎么办？

连续 90 天未登录将冻结账号。请联系管理员申请解冻。

## 技术相关

### 使用什么技术栈？

- 前端：Vue 3 + Vite + TypeScript
- 后端：Cloudflare Workers + D1
- 部署：Cloudflare Pages + Workers

### 开源协议是什么？

使用 AGPL-3.0 开源协议。
