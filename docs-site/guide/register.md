# 注册账号

本章介绍如何注册 Namio.World 账号。

## 注册流程

### 1. 访问注册页面

打开浏览器，访问 [Namio.World](https://nomio-dashboard.pages.dev)，点击"免费注册"按钮。

### 2. 填写注册信息

| 字段 | 说明 | 必填 |
|------|------|------|
| 用户名 | 二级域名前缀 | ✅ |
| 密码 | 至少 8 个字符 | ✅ |
| 确认密码 | 再次输入密码 | ✅ |

### 3. 提交注册

点击"注册"按钮完成注册。注册成功后会自动登录并跳转到仪表盘。

## 用户名规则

用户名将作为你的二级域名前缀，需要遵守以下规则：

### 允许的字符
- 小写字母：`a-z`
- 数字：`0-9`
- 连字符：`-`

### 长度要求
- 最短：3 个字符
- 最长：32 个字符

### 命名规则
- 不能以连字符开头或结尾
- 不能包含连续的连字符
- 注册后不可修改

### 示例

| 用户名 | 域名 | 状态 |
|--------|------|------|
| alice | alice.nomio.world | ✅ 有效 |
| bob123 | bob123.nomio.world | ✅ 有效 |
| my-app | my-app.nomio.world | ✅ 有效 |
| ab | - | ❌ 太短 |
| -alice | - | ❌ 以连字符开头 |
| alice- | - | ❌ 以连字符结尾 |
| Alice | - | ❌ 包含大写字母 |

## 安全建议

### 密码安全
- 使用至少 8 个字符
- 包含大小写字母、数字和特殊字符
- 不要使用常见的密码
- 不要在多个网站使用相同的密码

### 账号安全
- 定期修改密码
- 不要与他人分享账号
- 及时退出登录

## 常见问题

### 用户名已被注册
如果用户名已被注册，系统会提示"用户名已存在"。请尝试其他用户名。

### 注册失败
注册失败可能的原因：
- 网络连接问题
- 用户名不符合规则
- 密码不符合要求

### 忘记密码
目前暂不支持密码重置功能。如果忘记密码，请联系管理员。
