# 快速开始

只需四步即可开始使用 Nomio.World。

## 第一步：注册账号

1. 访问 [Nomio.World](https://nomio-dashboard.pages.dev)
2. 点击"免费注册"按钮
3. 填写注册信息：
   - **用户名** - 将作为你的二级域名前缀（如 `alice`）
   - **密码** - 至少 8 个字符
4. 点击"注册"按钮

::: tip 用户名规则
- 只能包含小写字母、数字和连字符
- 长度 3-32 个字符
- 不能以连字符开头或结尾
- 注册后不可修改
:::

## 第二步：注册域名

登录后，进入"域名管理"页面：

1. 点击"注册二级域名"
2. 填写源站地址：
   - 必须以 `https://` 开头
   - 不支持 IP 地址
   - 源站必须有效响应 HTTPS 请求
3. （可选）填写回源 Host
4. 点击"注册域名"

::: warning 源站要求
你的源站必须：
- 支持 HTTPS
- 有有效的 SSL 证书
- 能够正常响应 HTTP 请求
:::

## 第三步：验证源站

注册域名后，需要验证源站所有权：

1. 在源站创建文件：`/.well-known/nomio-verify.txt`
2. 文件内容为页面显示的验证 Token（格式：`nomio-verify=xxx`）
3. 点击"验证源站"按钮

::: tip 验证失败？
- 确保文件可通过 HTTPS 访问
- 确保文件内容完全匹配（无多余空格或换行）
- 确保源站没有重定向该路径
:::

## 第四步：注册邮箱

你还可以注册邮箱地址：

1. 进入"邮箱"页面
2. 点击"注册邮箱"
3. （可选）填写转发邮箱
4. 点击"注册邮箱"

注册成功后，你将获得 `your-name@nomio.world` 邮箱地址。

## 开始使用

完成以上步骤后：

- 访问 `your-name.nomio.world` 查看你的网站
- 使用 `your-name@nomio.world` 接收邮件
- 在控制台管理域名和邮箱
