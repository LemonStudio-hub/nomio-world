# 隐私政策

**最后更新日期：2026年7月15日**

## 1. 引言

欢迎使用 Nomio.World（以下简称"本服务"）。我们深知个人信息对您的重要性，并会尽全力保护您的个人信息安全可靠。本隐私政策适用于您通过 Nomio.World 平台提供的所有服务。

## 2. 信息收集

### 2.1 我们收集的信息

当您使用本服务时，我们可能会收集以下信息：

| 信息类型 | 说明 | 用途 |
|----------|------|------|
| 用户名 | 您注册时选择的标识 | 账号识别、域名分配 |
| 密码哈希 | 加密后的密码 | 账号安全认证 |
| 源站地址 | 您配置的网站地址 | 域名转发服务 |
| 邮件内容 | 发送到您邮箱的邮件 | 邮件托管服务 |
| 访问日志 | 服务使用记录 | 服务运维和安全 |

### 2.2 我们不收集的信息

我们**不会**收集以下信息：

- 真实姓名
- 身份证号
- 电话号码
- 个人地址
- 支付信息
- Cookie 和跟踪器

## 3. 信息使用

我们收集的信息仅用于以下目的：

1. **服务提供**：为您提供域名和邮箱服务
2. **账号安全**：保护您的账号免受未经授权的访问
3. **服务改进**：优化和改进我们的服务
4. **法律合规**：遵守适用的法律法规

## 4. 信息存储

### 4.1 存储位置

您的数据存储在 Cloudflare 的全球边缘网络中，包括：

- **D1 数据库**：用户信息、邮件元数据
- **Workers**：服务运行时

### 4.2 数据保留

- **账号数据**：在您删除账号前持续保留
- **邮件数据**：存储在您的邮箱中，您可以随时删除
- **日志数据**：保留 30 天后自动删除

## 5. 信息共享

我们**不会**将您的个人信息出售、交换或以其他方式转让给第三方，除非：

- 获得您的明确同意
- 法律法规要求
- 保护我们的合法权益

## 6. 数据安全

我们采取以下安全措施保护您的数据：

- **传输加密**：所有数据传输使用 HTTPS 加密
- **密码保护**：密码使用 PBKDF2 算法加密存储
- **访问控制**：严格的访问权限管理
- **安全审计**：定期进行安全审计

## 7. 您的权利

根据适用的数据保护法律，您享有以下权利：

- **访问权**：查看我们持有的您的个人信息
- **更正权**：更正不准确的个人信息
- **删除权**：请求删除您的个人信息
- **数据可携权**：以通用格式获取您的数据

如需行使上述权利，请通过 GitHub Issues 联系我们。

## 8. 儿童隐私

本服务不面向 13 岁以下的儿童。我们不会故意收集 13 岁以下儿童的个人信息。

## 9. 隐私政策更新

我们可能会不时更新本隐私政策。更新后的政策将在本页面发布，并注明更新日期。建议您定期查看本政策。

## 10. 联系我们

如果您对本隐私政策有任何疑问，请通过以下方式联系我们：

- **GitHub Issues**：[项目地址](https://github.com/LemonStudio-hub/nomio-world/issues)
